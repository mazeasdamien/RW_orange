# HRI Researcher Assistant

This app helps you analyze research papers using generic AI providers (e.g. Claude via Proxy).

View your app in AI Studio: https://ai.studio/apps/drive/1XxCbmvKbiotl1Cho3reDUiaNZT3fFJHR

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Run the app:
   `npm run dev`
